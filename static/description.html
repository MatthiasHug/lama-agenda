Traktanden-App mit SvelteKit

Ein einfaches Webtool zum Verwalten von Traktanden – Fokus auf CI/CD im Modul 324

Projektidee

Diese Agenda- bzw. Traktanden-App wurde im Rahmen des Moduls 324 erstellt. Der Fokus liegt nicht primär auf der Programmierung, sondern auf dem Aufbau einer modernen CI/CD-Umgebung. Die Anwendung dient als einfach verständliches Beispielprojekt.

1. Installation von SvelteKit und Projektstart

Projekt erstellen

npm create svelte@latest my-agenda-app

Im Wizard auswählen:

  • Skeleton project
  • JavaScript with JSDoc comments
  • ESLint
  • Playwright

Dependencies installieren

cd my-agenda-app
npm install

Projekt starten

npm run dev -- --open

2. Aufbau der Agenda-App

Die eigentliche Logik ist sehr einfach gehalten, damit der Fokus auf Deployment und Automatisierung liegt. Der Code kann in der Datei src/routes/+page.svelte ergänzt werden. 

<script>
let agenda = [];
let newAgenda = '';

function addAgenda() {
  if (newAgenda) {
    agenda = [...agenda, newAgenda];
    newAgenda = '';
  }
}
</script>

3. Start der Anwendung

Die App wird mit folgendem Befehl gestartet:

npm run dev

Danach ist sie im Browser erreichbar unter:

http://127.0.0.1:5173

Technologien

  • SvelteKit
  • JavaScript (mit JSDoc)
  • ESLint
